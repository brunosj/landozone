J’aime toujours voir quels outils les gens utilisent — voici les miens si vous êtes curieux.

### Utilitaires

- **Rectangle** : une de mes applis préférées sur Mac est [Rectangle](https://rectangleapp.com/), un gestionnaire de fenêtres qui permet d’organiser rapidement fenêtres et espaces de travail. Je passe sans cesse entre VS Code, le terminal, le navigateur, etc., et cet outil m’aide à le faire vite et proprement.
- **DeepL** : tout le monde connaît [DeepL](https://www.deepl.com/fr/macos-app), mais je recommande fortement l’app bureau : copier du texte dans le presse-papiers et le traduire avec un raccourci clavier.
- **Toggl Track** : j’utilise ce simple [suivi du temps](https://toggl.com/) pour mes heures de travail — interface claire et beaucoup de fonctions utiles.

### Outils web

- **Cursor** : Cursor avec [Claude](https://www.anthropic.com/en/chat-with-claude) est mon duo préféré pour coder et ça me simplifie vraiment la vie.
- **Uptime Kuma** : outil de [supervision auto-hébergé](https://github.com/louislam/uptime-kuma) pour sites et serveurs. Installation en quelques minutes et interface simple pour surveiller vos sites.
